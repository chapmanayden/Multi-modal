Computer Troubleshooting Flowchart

Follow the guide below to resolve your computer issue.

Basic Troubleshooting Steps

  1. Ensure the computer is powered on. If not, turn it on.
  2. Check if the monitor/display is on and connected.
  3. Try restarting the computer.
  4. Ensure all cables are securely connected.
  5. Test the computer on a different power outlet.
  6. Are there any unusual noises or lights from the computer? Note them down.
  7. Run a basic antivirus/malware scan.
  8. Free up storage space if the computer is running slow.
  9. Update your operating system and software.
  10. Reset network settings if facing internet issues.

Intermediate Troubleshooting Steps

  1. Try booting the computer in Safe Mode.
  2. Clear temporary files and cache.
  3. Update or reinstall device drivers.
  4. Test with an external monitor (for display issues).
  5. Perform a system restore to an earlier point.
  6. Check for overheating - clean dust from fans and vents.
  7. Run a disk check and memory check tool.
  8. Reconfigure or reset the BIOS/UEFI settings.

Advanced Troubleshooting Steps

  1. Inspect motherboard for visible damage or burnt components.
  2. Test individual hardware components (RAM, GPU, CPU).
  3. Check system logs for errors or warnings.
  4. Reinstall the operating system.
  5. Use bootable diagnostic tools to test hardware.
  6. Inspect PSU output using a multimeter.
  7. Check network configurations, firewall rules, and routing.
  8. Examine storage devices for hardware failures.
  9. Test with a different PSU or motherboard.
  10. Seek expert advice or consider replacement of faulty hardware.
